# MES-Manufacturing-Execution-System
基于Springboot+Vue的MES生产制造执行系统，专为生产制造行业设计。后端采用Springboot框架，提供强大的数据处理能力；前端使用Vue框架，构建直观易用的操作界面。该系统涵盖生产计划排程、仓储管理、制造执行、质量管理等功能模块，实现生产过程的全面数字化管理，帮助企业提升生产效率、优化资源配置，并确保产品质量。
